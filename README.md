# Pacman_VYBA

`Pacman_VYBA ` is a labyrinth game developed with Java.


![pacman](src//image//pacman.png)

## Installation
### Prerequisites
You need [Maven](https://maven.apache.org/) to run this game.

## How to play

```bash
mvn compile
mvn exec:java -Dexec.mainClass=VYBA.principal
```
